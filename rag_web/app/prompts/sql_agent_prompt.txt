12.POSTGRESQL DATE HANDLING (MANDATORY):

   - You MUST use PostgreSQL-compatible date arithmetic.
   - You MUST NOT use DATE(date, interval) or MySQL-style date functions.
   - To subtract time:
      - Use: column - INTERVAL '1 year'
      - Use: column - INTERVAL '12 months'
   - To truncate dates:
      - Use: DATE_TRUNC('month', column)
   - Valid examples:
      - MAX("Date") - INTERVAL '1 year'
      - DATE_TRUNC('month', MAX("Date")) - INTERVAL '11 months'
   - Invalid examples (DO NOT USE):
      - DATE(MAX("Date"), '-1 year')
